# 🎓 Learning Management System (LMS)

A simple and responsive **Learning Management System (LMS)** built using **HTML, CSS, and JavaScript**. This project demonstrates the basic functionality of an online learning platform where users can browse courses, search for courses, track learning progress, and switch between light and dark themes.

---

## 🚀 Features

* 📚 Display available courses
* 🔍 Search courses by name
* ✅ Mark courses as completed
* 📊 Dynamic progress bar
* 🌙 Light/Dark mode toggle
* 📱 Responsive design for desktop and mobile
* 🎨 Clean and modern user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📖 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it using any modern web browser.
4. Start exploring the Learning Management System.

No installation or external libraries are required.

---

## 💻 Functionalities

### 📚 Course List

Displays multiple learning courses with a short description.

### 🔍 Search Courses

Allows users to search courses instantly by typing in the search box.

### ✅ Course Completion

Users can mark any course as completed by clicking the **Complete** button.

### 📊 Progress Tracking

The progress bar automatically updates as more courses are completed.

### 🌙 Dark Mode

Users can switch between Light Mode and Dark Mode for a better viewing experience.
